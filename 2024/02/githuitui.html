git回退本地分支与远程仓库

 2024/02/07

本地分支回退

先用下面命令找到要回退的版本的commit id：
git log --oneline

要退出 git log 命令的输出，可以按下键盘上的 “q” 键

接着回退版本:
git reset 3e47f0a --hard
3e47f0a就是你要回退的版本的commit id

远程仓库回退

先进行本地回退
然后强制推送到远程分支：

git push -f
